## サイト概要
商品機能、ジャンル機能、カート機能、注文機能、注文商品機能、ログイン・サインアップ機能、コメント機能があります。
カスタマー側で商品追加、配送先追加、カート利用、注文購入、商品にコメントができます。
管理者側でジャンル機能を使用できます。
